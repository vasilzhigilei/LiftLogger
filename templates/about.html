LiftLogger


LiftLogger is a free website allowing users to log compound weightlifting personal records

Created by Vasil Zhigilei. Source code may be viewed on GitHub

On the technical side of things, LiftLogger uses Golang with the Gorilla HTTP framework for it's backend. User data is stored in Postgres, while sessions are stored in a faster Redis cache. For the frontend, Bootstrap 4 is used, with the frontend using vanilla Javascript (& a little jQuery)

Login and registration is achieved using Google's OAuth2 authentication platform
